# fleet-ui 组件库

### 快速开始

#### 1、安装组件库

````bash
 npm i fleet-ui
````

#### 2、引用组件库
````javascript
    // 全部引入 
    import 'fleet-ui/dist/css/index.css'
    import fleetUi from 'fleet-ui'
    Vue.use(fleetUi)

    // 按需引用
    import 'fleet-ui/dist/css/card.css'
    import { Card } from 'fleet-ui'
    Vue.use(Card)
````
